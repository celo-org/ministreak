# UI Revamp Implementation Plan

> Branch: `ui-revamp` (one branch for the whole revamp). Visual source of truth:
> `docs/superpowers/specs/2026-07-13-ui-revamp-mockup.html`. Design: `docs/superpowers/specs/2026-07-13-ui-revamp-design.md`.

**Goal:** Restyle the app to the approved "MiniQuiz-chunk / MiniStreak-soul" look, add the 3rd (Me/profile) tab, and replace every emoji with brand-sketch SVG icons — no backend/contract change.

**Execution note:** This is visual work — "done" means it *looks* like the mockup, not just that tests pass. Each task ends with **controller-led visual verification** (run the app / render the screen and compare to the mockup) plus `npm run type-check` + suite green, then a commit. Icons get refined against the render, not authored blind-and-forgotten. Order matters: foundation → icons → nav/profile → screens → onboarding → sweep.

## Global constraints
- No emoji anywhere (icons = monochrome filled brand-sketch SVGs, `fill=currentColor`).
- Keep all existing logic/hooks (entry, refund, resolve, optimistic today, oracle, KV). Restyle only.
- Truthful copy (XP = daily transaction). "Welcome back" only when `useProfile().profile.xp > 0`.
- Extend the existing token/tailwind system; don't fork. Commit per task. From `frontend/`.

---

### Task 1 — Foundation: font, tokens, primitives
- **Font:** in `app/layout.tsx` add `Fredoka` from `next/font/google` (`variable:"--font-fredoka"`, weights 400–700); in `tailwind.config.ts` set `fontFamily.display` → `["var(--font-fredoka)", ...]` (rounded), keep `sans`=DM Sans, `mono`=JetBrains. Apply the Fredoka variable class on `<body>`.
- **Tokens:** in `globals.css :root` add the candy accents + tints (`--amber/--sky/--berry/--gold` + `*-tint`, values from the design doc); add matching colors to `tailwind.config` `theme.extend.colors`.
- **Primitives:** in `globals.css @layer components`, author the classes from the mockup verbatim in feel — `stat`, `chip`, `pill-stat`, `toggle`, `btn-primary` (chunky forest + solid drop-shadow), `avatar`, `lvl`, `hero`, `podium`, `bar`, `lbrow`, `daily`, and refresh `card`. Reuse existing where present.
- **Verify:** app builds; a scratch page (or Home) shows the new button/card/chip render correctly; type-check clean. Commit `foundation: rounded font, accent tokens, chunky primitives`.

### Task 2 — Brand-sketch icon components
- Create `frontend/components/icons/` with monochrome filled SVG components: `StreakIcon`, `ScoreIcon`, `FreezeIcon`, `TrophyIcon`, `CrownIcon`, `MedalIcon`, `MeIcon`. Base shapes from the mockup; then **refine against a render** to match the hand-sketch weight of `home.svg`/`leaderboard.svg`.
- **`MeIcon`:** derive from `public/leaderboard.svg` — strip its 3-figure path to a single figure; save both a `MeIcon` component and a `public/me.svg` for the nav. Keep `public/home.svg`, `public/leaderboard.svg`.
- **Verify:** render an icon gallery (scratch/Storybook-lite page or the style-tile), eyeball against the two real assets, iterate until consistent. Commit `icons: brand-sketch icon set + me figure`.

### Task 3 — 3-tab BottomNav
- `components/BottomNav.tsx`: `NAV_ITEMS = Home(/) · Board(/leaderboard) · Me(/me)`; Board keeps `leaderboard.svg`, Me uses `me.svg`/`MeIcon`. Rounded label font, forest active. Verify all three tabs render + active states. Commit.

### Task 4 — `/me` profile page
- New `app/me/page.tsx` (client): the profile per mockup — avatar (monogram from address via `pseudonymFor`/initials) + `LVL n`, pseudonym + `shortAddress`, 2×2 stat cards (Streak/Score/Freezes/Rank) + XP progress card. Data from `useProfile`, `usePlayerStats`, `useLeaderboard` (rank), `useAccount`. Not-connected → connect prompt (reuse `WalletBadge`). Verify render vs mockup. Commit.

### Task 5 — Home restyle (`app/page.tsx`)
- Masthead; header row with avatar + greeting (**"Welcome back" only if `profile.xp>0`**, else just name; none if not connected) + a streak `pill-stat`; **pot hero** (gradient, round #, pot, players, countdown from `RoundTimer` data, "Enter this week · 0.10 USDT" → existing `EntryButton` logic); **Daily XP card** (7 dots from the round-day windows + today's `+xpForDay(streak+1)` + "Send a transaction daily to earn XP"); restyled streak card slot; **Top 5** via `lbrow`. Preserve refund/resolve/entry/optimistic logic. Update copy truthfully. Verify. Commit.

### Task 6 — Board restyle (`app/leaderboard/page.tsx` + `components/Leaderboard.tsx`)
- Podium hero (top-3, crown, elevated winner, monogram avatars), `This week / Last week` toggle (visual; wire only if data exists), `lbrow` rows (rank/medal · avatar · name+@addr·Lv · score · est. prize), highlight connected player. Replace 🥇🥈🥉 with `MedalIcon`/rank number. Update `Leaderboard.test.tsx`. Verify. Commit.

### Task 7 — StreakCard restyle (`components/StreakCard.tsx`)
- Chunky card, flame `chip`, optimistic pill, **freeze count with `FreezeIcon`** (remove 🛡), Level/XP surface. Update `StreakCard.test.tsx` (drop emoji assertions, add icon presence). Verify. Commit.

### Task 8 — Onboarding icons (`components/OnboardingCarousel.tsx`)
- Slide 1 → `Logo_Color.svg`; slides 2/3/4 → brand-sketch icons (ticket/coin, trophy, shield). Remove 🔥🎟🏆🛡; keep copy + behavior. Update tests if they asserted glyphs. Verify. Commit.

### Task 9 — Emoji sweep + final QA
- Grep the app for any remaining emoji; replace/remove. Full visual pass across Home/Board/Me/Onboarding on a phone viewport. `npm test` + `npm run type-check` + `npm run lint` clean. Commit `chore(ui): emoji sweep + final polish`.

## Final
- Full suite green, type-check clean, visual QA done → finish the `ui-revamp` branch (PR to main).
